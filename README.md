# 📊 HR Analytics Dashboard | Power BI

An interactive **HR Analytics Dashboard** built in **Microsoft Power BI** to help HR teams analyze workforce demographics, employee performance, promotion eligibility, attrition-related indicators, and overall workforce distribution.

This project demonstrates how Power BI can transform raw HR data into actionable business insights through interactive visualizations, KPIs, and drill-down analysis.

---

## 📸 Dashboard Preview

### 🏠 Home Dashboard
> Provides a high-level overview of the organization's workforce.

- Total Employees
- Male vs Female Employees
- Employees Due for Promotion
- Employees Not Due for Promotion
- Employees Eligible to be Retrenched
- Employees Currently in Service
- Service Year Distribution
- Job Level Distribution
- Distance Status Analysis
- Interactive Filters

---

### 🔍 Deep Insights Dashboard
> Detailed HR insights for better workforce planning.

- Employees eligible for both Retrenchment & Promotion
- Job Satisfaction Analysis
- Overtime Analysis
- Job Role-wise Employee Summary
- Promotion Status by Job Role
- Retrenchment Eligibility by Job Role
- Interactive Cross Filtering

# 📈 Key KPIs

| KPI | Description |
|------|-------------|
| Total Employees | Overall workforce count |
| Male Employees | Total male employees |
| Female Employees | Total female employees |
| Due for Promotion | Employees eligible for promotion |
| Not Due for Promotion | Employees not yet eligible |
| On Service Employees | Employees currently active |
| Eligible to be Retrenched | Employees meeting retrenchment criteria |
